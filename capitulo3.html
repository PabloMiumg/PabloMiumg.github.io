Tesina capitulo III


¿Que es el marco metodologico?

Conjunto de métodos que se siguen en una investigación científica o en una exposición doctrinal.

Es el “conjunto de pasos, técnicas y procedimientos que se emplean para formular y resolver problemas”. Este método se basa en la formulación de hipótesis las cuales pueden ser confirmadas o descartadas por medios de investigaciones relacionadas al problema.

¿Como se realiza una hipotesis y para que sirve?

Una hipótesis es una proposición o suposición tentativa que se formula con el propósito de explicar un fenómeno observado o establecer una relación entre variables en un estudio de investigación. La formulación de una hipótesis es una etapa clave en el proceso de investigación científica y sirve para orientar y estructurar la indagación.

Pasos para formular una hipotesis

  • Identificar el problema o fenómeno de interés: Se debe tener claro el tema de investigación y el problema específico que se pretende abordar.
  • Realizar una revisión bibliográfica: Es importante revisar fuentes relevantes, como artículos científicos, libros y estudios previos, para conocer el estado actual del conocimiento sobre el tema. Esto ayuda a identificar las lagunas en la investigación existente y establecer una base sólida para la formulación de la hipótesis.
  • Definir variables: Se deben identificar las variables que están involucradas en el problema de investigación. Las variables son características, conceptos o factores que se estudian y que pueden variar. Por ejemplo, en un estudio sobre el efecto de un nuevo medicamento en la reducción del dolor, las variables pueden ser el medicamento (variable independiente) y el nivel de dolor (variable dependiente).
  • Establecer la relación entre variables: La hipótesis debe plantear una posible relación entre las variables identificadas. Por ejemplo, se puede formular una hipótesis nula (afirma que no hay relación entre las variables) o una hipótesis alternativa (sugiere que hay una relación entre las variables).
  • Ser específico y verificable: La hipótesis debe ser clara, precisa y verificable mediante datos y evidencia empírica. Debe ser lo suficientemente específica para permitir su evaluación y prueba durante el estudio de investigación.
