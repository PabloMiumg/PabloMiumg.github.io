Teoria de conjuntos


Es la rama de la logica matematica que estudia las propiedades, las operaciones y las relaciones de los conjuntos.

Notacion

  • letras mayusculas, como A,B,C para representar los conjuntos y letras minusculas para los elementos

  • puede designarse enumerando sus elementos dentro de llaves

  • los elementos se separan por comas, el orden no importa, no se pueden repetir elementos

    • relaciones entre conjuntos

  • si C,D son conjuntos universo(U) decimos que C es subconjunto de D y escribimos C=D, si cada elemento de C es un elemento de D.

    • Subconjunto propio0

    si ademas, D tienen un elemento que no esta en C, entonces C es un subconjunto propio de D y es CcD.

    propiedades de los subconjuntos

  • reflexiva:Todo conjunto es subconjunto de si mismo

  • antisimetrica:si A es subconjunto de B y B es subconjunto de A entonces A es igual a B

  • transitiva:si A es subconjunto de B y B es subconjunto de C entonces A es subconjunto de C

  • conjunto vacio como subconjunto: al no tener elementos el conjunto vacion se encuentra presente como parte de cuaquier agrupacion de conjuntos

    es la sucesion finita de cada premisa y la conclusion, una forma argumentativa es valida si la conclusion se deriva de sus premisas.


  • una conclusion derivada de las premisas se llama consecuencia logica
  • asi, la forma argumentativa sera valida solo si, la conclusion es derivada se sus premisas
  • el termino medio debe estar al menos una vez en la premisa
